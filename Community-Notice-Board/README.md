# Community Notice Board

A small web project that displays community notices in a clean, card-based layout.

## What this project is

A community notice board built with HTML, CSS, and JavaScript. It shows notice cards with titles, descriptions, and categories so users can browse announcements easily.

## What it currently does

- Displays a list of notice cards
- Uses basic JavaScript to render notices dynamically
- Includes simple styling for a readable layout

## Planned features

- Filter buttons at the top to filter notices by category
- Search bar that filters notices by keyword in real time
- Smooth card animations with CSS transitions
